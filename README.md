Currency Converter
Автор: Халидова Хадиджа Ренатовна

Описание проекта
Графическое приложение для конвертации валют с использованием внешнего API и сохранением истории операций.

Технологии
Python 3.x.
Tkinter (стандартная библиотека).
Requests (для работы с API).
JSON (для хранения истории).
Git (для контроля версий).

Установка и запуск
1. Установите Python (если не установлен).
2. Установите зависимости:
pip install requests
3. Замените YOUR_API_KEY в Main.py на ваш ключ от exchangerate-api.com.
Запустите приложение:
python Main.py

Как получить API-ключ
1. Перейдите на сайт exchangerate-api.com.
2. Зарегистрируйтесь и получите бесплатный API-ключ.
3. Вставьте ключ в переменную API_URL в файле Main.py вместо YOUR_API_KEY.

Использование приложения
1. Выберите валюту «Из» и «В».
2. Введите сумму для конвертации.
3. Нажмите «Конвертировать».
4. Результат появится под кнопкой.
5. История операций отображается в таблице и сохраняется в файл history.json.

Примеры использования (тесты)

Позитивный тест
Ввод: USD → EUR, сумма: 100.
Ожидаемый результат: Конвертация выполнена, результат > 0.

Негативный тест
Ввод: USD → EUR, сумма: -50.
Ожидаемый результат: Ошибка «Сумма должна быть больше 0».

Граничный тест
Ввод: USD → EUR, сумма: 0.
Ожидаемый результат: Ошибка «Сумма должна быть больше 0».
